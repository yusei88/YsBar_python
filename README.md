# Features
家族にカクテルを作ることがあるので、そのオーダーをとるシステムを作ることで、実際に使われるような技術の練習になると考えた。

# Usage
注文者がメニューリストから注文ボタンを押すと選んだドリンクが注文リストに追加される。
